5.2 Key Concepts

Figure 5.1: Dataset verbs are functions that require a dataframe as the primary argument, perform some transformation, then present a new dataframe as the return value.

5.3 The dplyr Package

This chapter will demonstrate a few basic dataset functions contained within the dplyr package. There are a few things to note as you get started:

  • dplyr functions (dataset verbs) accept a dataset as the argument, transform the data, and return a new dataset.
  • The first argument is always the dataset name, and columns can be referenced directly by name without quotation marks.
  • dplyr functions will return data as a “tibble” (“tbl_df” class), which is just a data table that has been nicely formatted for printing.

taxes.owed <- c(0,9353,0,0,1546,0,0,0,634,0,362,0,0)

5.5 Arrange() (sort datasets)

We use the arrange() function to sort a dataset by one or more columns. By default, it sorts from smallest to largest.

arrange( PlantGrowth, weight )
weight group
3.59 trt1
3.83 trt1
4.17 ctrl
4.17 trt1
4.32 trt1

If we prefer the dataset be sorted from largest to smallest, we can add the descending desc() function to the sort variable:

arrange( PlantGrowth, desc(weight) )
weight group
6.31 trt2
6.15 trt2
6.11 ctrl
6.03 trt1
5.87 trt1

Or alternatively we can get an equivalent result by using the shortcut syntax of adding a negative in front of the variables that we want sorted from largest to smallest:

arrange( PlantGrowth, -weight )

We can sort by multiple columns at once:

arrange( PlantGrowth, group, weight )
group weight
ctrl 5.17
ctrl 5.18
ctrl 5.33
ctrl 5.58
ctrl 6.11
trt1 4.69
trt1 4.81
trt1 4.89
trt1 5.87
trt1 6.03

NOTE, the equivalent core R functions would use order() and subset[] functions together. You might see examples written like this:

PlantGrowth[ order(PlantGrowth$weight, decreasing=TRUE) , ]
  weight group
21 6.31 trt2
28 6.15 trt2
4 6.11 ctrl
17 6.03 trt1
15 5.87 trt1
29 5.8 trt2

As you can see, the dplyr versions are typically more intuitive and concise.
